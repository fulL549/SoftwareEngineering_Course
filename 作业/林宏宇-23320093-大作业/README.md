# AI-EduMate 智能教育助手

## 1. 项目简介
AI-EduMate 是一款面向学生群体和科研工作者的智能学习效率工具。它深度融合了人工智能技术，旨在提供“计划-学习-整理-复习”的全流程闭环支持。通过实现智能化、个性化、无缝化的学习模式，AI-EduMate 能够主动理解用户学习需求，并基于认知特征动态调整学习策略，从而帮助用户高效管理学习任务、整理知识资料，并消除多工具切换带来的不便，全面提升学习效率和体验。


## 2. 核心功能

### 用户管理
- 提供安全的用户注册、登录、信息更新和注销功能
- 支持JWT身份认证和登录状态持久化

### 智能问答 (AI聊天)
- 与AI助手进行多轮对话，支持Markdown格式渲染，包括代码高亮和复制功能
- 实时显示“正在输入”状态，优化用户体验
- 自动管理对话历史，并可创建新对话

### 学习工具集成
- **课程表管理**：查看、添加、编辑和删除个人课程表
- **预约行程管理**：规划和管理个人预约事件
- **笔记本功能**：在线记录、编辑、分类和检索笔记
- 所有学习工具模块均支持数据实时刷新

### 用户界面与交互
- 全平台响应式设计，适配桌面、平板和移动设备
- 可折叠侧边栏，提供清晰的用户信息展示和快捷菜单导航
- 友好的模态对话框交互


## 3. 技术栈
本项目采用MCP (Model-Controller-Persistence)架构进行设计与实现，确保了模块化、高内聚和低耦合。

### 后端开发
- **语言与框架**：Java, Spring Boot
- **AI集成**：DeepSeek大模型
- **数据库**：MySQL

### 前端开发
- **基础技术**：HTML, CSS, JavaScript
- **库/框架**：SockJS, Stomp.js (WebSocket), Marked.js (Markdown解析), Highlight.js (代码高亮)

## 6. 如何运行

### 后端部署
1. 确保已安装Java开发环境 (JDK) 和Maven
2. 配置MySQL数据库，并导入相应的数据库脚本
3. 在后端项目根目录执行Maven命令进行构建和运行
4. 确保后端服务在http://127.0.0.1:8081/端口上成功启动

### 前端部署
1. 确保已安装Node.js和npm/yarn (如果前端项目有依赖管理)
2. 打开前端项目文件 (例如index.html或通过本地服务器启动)
3. 确保前端能够正确访问后端接口http://127.0.0.1:8081/
